# 📉 Análise de Desemprego e Inflação no Brasil

Projeto final da disciplina de Ciência de Dados com Python — Inatel, 2025.

Análise exploratória de dados socioeconômicos brasileiros, investigando
a relação entre desemprego e inflação ao longo do tempo.

## Tecnologias

- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Jupyter Notebook

## Dados utilizados

- `brazil_unemployment.csv` — série histórica de desemprego no Brasil
- `brazil_inflation.csv` — índices de inflação brasileiros
- `global_inflation_data.csv` — inflação global para comparação

## Como executar

```bash
pip install -r requirements.txt
jupyter notebook final_analysis.ipynb
```
